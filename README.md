# Personal Assistant

When you deploy the app in Streamlit, add the following secrets:


```
OPENAI_API_KEY = "sk-xxx"

[cookie]
expiry_days = 30
key = "some_signature_key"
name = "some_cookie_name"

[credentials]
usernames = {your_user_name_goes_here = {name = "", password = "your_password_goes_here"}}
```
